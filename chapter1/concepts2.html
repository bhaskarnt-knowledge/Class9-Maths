1. Locating Irrational Numbers on the Number Line

Visualizing the placement of irrational numbers like $\sqrt{2}$, $\sqrt{5}$, and $\sqrt{13}$ on the number line using the Pythagorean theorem.

This animation would demonstrate the construction of $\sqrt{13}$ as described in the chapter.

Example: Locating $\sqrt{13}$

Imagine an animated GIF showing the steps:

  1. Draw a number line.
  2. Mark OA = 3 units from the origin (O).
  3. Draw AB = 2 units perpendicular to OA at A.
  4. Connect OB. By Pythagoras, $OB = \sqrt{3^2 + 2^2} = \sqrt{9+4} = \sqrt{13}$.
  5. Use a compass with center O and radius OB to draw an arc intersecting the number line at C. Point C represents $\sqrt{13}$.

3. Operations on Real Numbers: Visualizing Simplification

Using animated steps to show the simplification of expressions involving radicals, making the distributive property and rationalization clearer.

Example: $(3\sqrt{5}-5\sqrt{2})(4\sqrt{5}+3\sqrt{2})$

An animation could highlight each step of the multiplication and simplification process, as shown in Sample Question 3.


        (3√5 - 5√2)(4√5 + 3√2)
        = 3√5 * (4√5 + 3√2) - 5√2 * (4√5 + 3√2)  <-- Highlight distributive property
        = (12 * 5) + (9√10) - (20√10) - (15 * 2) <-- Highlight individual multiplications
        = 60 + 9√10 - 20√10 - 30               <-- Combine terms
        = 30 - 11√10                         <-- Final simplified form

Example: Rationalizing the Denominator $\frac{6}{3\sqrt{2}-2\sqrt{3}}$

An animation could show the multiplication by the conjugate and the subsequent simplification, as shown in Sample Question 4.


        (6 / (3√2 - 2√3)) * ((3√2 + 2√3) / (3√2 + 2√3)) <-- Highlight multiplication by conjugate
        = 6(3√2 + 2√3) / ((3√2)^2 - (2√3)^2)          <-- Highlight difference of squares
        = 6(3√2 + 2√3) / (18 - 12)
        = 6(3√2 + 2√3) / 6
        = 3√2 + 2√3
